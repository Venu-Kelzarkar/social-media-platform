# Social Media Platform Using Django

## Description
This is a simple social media web application developed using Django. Users can create posts, view posts, and like posts. The project demonstrates basic web development using Python and Django.

## Features
- Create Post
- View Posts
- Like Posts
- Admin Panel
- User Login and Logout

## Technologies Used
- Python
- Django
- HTML
- SQLite
- VS Code

## How to Run

```bash
python manage.py migrate
python manage.py runserver
